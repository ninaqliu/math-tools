Simple · Compound · Continuous Interest

A = P(1+rt)  ·  A = P(1+r/n)ⁿᵗ  ·  A = Peʳᵗ

Read me first Three types of interest compared ▾

Simple Interest — A = P(1 + rt)

  • Interest is calculated only on the original principal — never on accumulated interest
  • Growth is linear — the graph is a straight line
  • Interest earned each year = P × r (constant)

Compound Interest — A = P(1 + r/n)ⁿᵗ

  • Interest is added to the principal, then earns interest itself — exponential growth
  • n = compounding frequency: 1=annual, 2=semi-annual, 4=quarterly, 12=monthly, 365=daily
  • More compounding periods = slightly more growth

Continuous Compounding — A = Peʳᵗ

  • The mathematical limit as n → ∞ — interest compounded every instant
  • e ≈ 2.71828 (Euler's number)
  • The difference between daily (n=365) and continuous is very small in practice
  • Rule of 72: divide 72 by the rate % to estimate doubling time

Key comparison

  • All three use the same P and r — only the growth pattern differs
  • Simple < Compound < Continuous (for the same P, r, t)
