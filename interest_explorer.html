Formula

A = P(1 + rt)  ·  A = final amount, P = principal, r = annual rate, t = time in years

Key idea

  • Interest is calculated only on the original principal — never on accumulated interest
  • Growth is linear — the graph is a straight line
  • Interest earned each year = P × r (constant)

Formula

A = P(1 + r/n)^(nt)  ·  n = number of compounding periods per year

Key idea

  • Interest is added to the principal, then earns interest itself — exponential growth
  • More compounding periods = more growth
  • n = 1 (annual), n = 2 (semi-annual), n = 4 (quarterly), n = 12 (monthly), n = 365 (daily)

  • Compare n = 1 vs n = 365 — how much does daily compounding add?

Formula

A = Pe^(rt)  ·  e ≈ 2.71828 (Euler's number)

Key idea

  • The mathematical limit of compounding — as n → ∞, the formula becomes Pe^(rt)
  • Gives the absolute maximum growth for a given rate
  • The difference between daily (n=365) and continuous is very small in practice

Formula

PV = FV / (1 + r)^t  ·  PV = present value, FV = future value, r = rate, t = years

Key idea

  • Money today is worth more than the same amount in the future (time value of money)
  • PV asks: how much do I need to invest today to reach a target future value?
  • Higher rate or longer time = lower present value (you need less today)
  • The curve is a decreasing function — decay as time increases

Try this

  • Set FV = $10,000 at r = 5% for 10 years — what is the PV?
  • Increase r — does PV go up or down?
  • PV × (1+r)^t should equal FV — verify this!
