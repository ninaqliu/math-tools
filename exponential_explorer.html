Exponential Functions

y = a · bˣ

Read me first What this tool shows & how to use it ▾

What this shows

This tool explores y = a · bˣ. Select a category tab then adjust sliders to see how a and b shape the curve.

Categories

  • b > 1, a > 0 — exponential growth. Curve rises as x increases.
  • 0 < b < 1, a > 0 — exponential decay. Curve falls toward 0 as x increases.
  • b > 1, a < 0 — reflected growth. Curve falls below x-axis.
  • 0 < b < 1, a < 0 — reflected decay. Curve rises toward 0 from below.

Key concepts

  • y-intercept is always (0, a) since b⁰ = 1
  • Horizontal asymptote: y = 0 — the curve approaches but never touches the x-axis
  • The graph exists in all four quadrants — for negative x the curve is close to zero
  • Red dot = y-intercept (0, a)  ·  purple dashed line = horizontal asymptote y = 0

Try this

  • Set b = 2 — classic doubling growth
  • Set b = 0.5 — classic half-life decay
  • Change a — the y-intercept moves but the asymptote stays at y = 0
  • Watch how the curve behaves in the left half (x < 0)
