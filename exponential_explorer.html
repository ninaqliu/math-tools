What this shows

This tool explores y = a · bˣ. Select a category tab then adjust sliders to see how a and b shape the curve.

Categories

  • b > 1, a > 0 — exponential growth. Curve rises as x increases.
  • 0 < b < 1, a > 0 — exponential decay. Curve falls toward 0 as x increases.
  • b > 1, a < 0 — reflected growth. Curve falls below x-axis.
  • 0 < b < 1, a < 0 — reflected decay. Curve rises toward 0 from below.

Key concepts

  • y-intercept is always (0, a) since b⁰ = 1
  • Horizontal asymptote: y = 0
  • The curve never touches the x-axis
  • Red dot marks the y-intercept (0, a)

  • Set b = 0.5 with 0 < b < 1 — classic half-life decay
