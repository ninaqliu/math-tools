How to use this tool  ▾

What this shows

Set a numerator and denominator with the sliders. The tool displays the fraction using a proper fraction bar, classifies it, and draws a visual circle model. Use the Conversion section to see step-by-step working.

Key concepts

  • Proper fraction — numerator < denominator. Value is less than 1. Example: ³⁄₄
  • Improper fraction — numerator ≥ denominator. Value is 1 or greater. Example: ⁷⁄₄
  • Mixed number — a whole number plus a proper fraction. Example: 1¾
  • Improper fractions and mixed numbers represent the same value, written differently.

Try this

  • Set numerator = denominator — what kind of fraction is that? What value is it?
  • Make the numerator much larger than the denominator — how many whole circles appear?
  • Switch to the Conversion tab and adjust the sliders to see step-by-step working.
