Quadratic Functions

y = a[k(x − d)]² + c

Read me first What this tool shows & how to use it ▾

What this shows

This tool explores vertex form: y = a[k(x − d)]² + c. Adjust the sliders to see how each parameter transforms the parabola in real time.

Parameters

  • a — vertical stretch/flip. |a| > 1 narrows the parabola, |a| < 1 widens it. Negative a flips it downward (opens down).
  • k — horizontal stretch/compression. |k| > 1 compresses horizontally, |k| < 1 stretches it.
  • d — horizontal shift. Moves the vertex left (d < 0) or right (d > 0).
  • c — vertical shift. Moves the vertex up (c > 0) or down (c < 0).

Key concepts

  • Vertex is always at (d, c) — the turning point of the parabola
  • Axis of symmetry: x = d (shown as a dashed vertical line)
  • Faint grey curve shows the base parabola y = x² for comparison
  • Red dot = vertex  ·  Green dots = x-intercepts (roots), if they exist

Try this

  • Set a < 0 — which way does the parabola open?
  • Move d left and right — what travels with it?
  • Move c below zero with a > 0 — how many x-intercepts appear?
  • Make |k| very large — what happens to the width?
