What this shows

This tool explores the slope-intercept form: y = mx + b. Adjust the sliders to see how each parameter changes the graph in real time.

Parameters

  • m — slope. Positive = rising left to right, negative = falling, zero = horizontal line.
  • b — y-intercept. Where the line crosses the y-axis. The point (0, b) is always on the line.

Key concepts

  • Rise/run triangle is drawn to show slope visually
  • Red dot = y-intercept  ·  Green dot = x-intercept
  • x-intercept = −b/m (when m ≠ 0)
  • Parallel lines have equal slopes
  • Perpendicular lines: slopes multiply to −1

Try this

  • Set m = 0 — what happens to the line?
  • Set b = 0 — does the line pass through the origin?
  • Make m negative — which direction does the line slope?
