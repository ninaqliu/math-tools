What this shows

This tool explores the slope-intercept form of a linear function: y = mx + b. Adjust the sliders to see how each parameter changes the graph in real time.

Parameters

  • m — slope. Controls steepness and direction. Positive = rising, negative = falling, zero = horizontal.
  • b — y-intercept. Where the line crosses the y-axis. The point (0, b) is always on the line.

Key concepts

  • Rise/run triangle is drawn on the graph to show slope visually
  • Red dot = y-intercept (0, b)
  • Green dot = x-intercept where the line crosses the x-axis
  • x-intercept = −b/m (when m ≠ 0)
  • Parallel lines have equal slopes · Perpendicular lines have slopes that multiply to −1

Try this

  • Set m = 0 — what happens to the line?
  • Set b = 0 — does the line pass through the origin?
  • Make m negative — which way does the line slope?
  • What happens when m is very large or very small?
