Linear Functions

y = mx + b

Read me first What this tool shows & how to use it ▾

What this shows

This tool explores slope-intercept form: y = mx + b. Adjust the sliders to see how each parameter changes the graph in real time.

Parameters

  • m — slope. Positive = rising left to right. Negative = falling. Zero = horizontal line.
  • b — y-intercept. Where the line crosses the y-axis. The point (0, b) is always on the line.

Key concepts

  • A rise/run triangle is drawn to show slope visually
  • Red dot = y-intercept  ·  Green dot = x-intercept
  • x-intercept = −b/m (only when m ≠ 0)
  • Parallel lines share the same slope
  • Perpendicular lines have slopes that multiply to −1

Try this

  • Set m = 0 — what happens to the line?
  • Set b = 0 — does the line pass through the origin?
  • Make m negative — which direction does the line slope?
  • Set m = 2 and m = −½ — are those lines perpendicular?
