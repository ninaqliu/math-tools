What this shows

Every value can be written as a fraction, a decimal, or a percent — they are all equivalent. Use the three tabs to convert between any pair, see step-by-step working, and visualise the value on a bar.

Key rules

  • Fraction → Decimal: divide numerator ÷ denominator
  • Decimal → Percent: multiply by 100 (move decimal 2 places right)
  • Percent → Fraction: write over 100, then simplify
  • Fraction → Percent: convert to decimal first, then × 100

Try this

  • Set numerator = 1, denominator = 4 — classic ¼ = 0.25 = 25%
  • Try denominator = 3 — notice the repeating decimal
  • Enter 0.1 in the Decimal tab — see why 10% = 1/10
