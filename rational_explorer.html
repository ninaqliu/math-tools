Rational / Reciprocal Functions

y = a / [k(x − d)] + c

Read me first What this tool shows & how to use it ▾

What this shows

This tool explores the rational (reciprocal) function. The numerator is a and the denominator is k(x − d), producing a hyperbola with two branches.

Parameters

  • a — numerator. Larger |a| = branches stretch further from centre. Negative a flips both branches to opposite quadrants.
  • k — denominator factor. Larger |k| = branches compress toward the asymptote.
  • d — horizontal asymptote shift. Moves the vertical asymptote left or right.
  • c — vertical asymptote shift. Moves the horizontal asymptote up or down.

Key concepts

  • Vertical asymptote: x = d — denominator = 0 here, curve undefined
  • Horizontal asymptote: y = c — curve approaches this as x → ±∞
  • Two branches, one in each pair of opposite quadrants from centre (d, c)
  • Domain: all x ≠ d  ·  Range: all y ≠ c

Try this

  • Move d — watch the vertical asymptote shift
  • Move c — watch the horizontal asymptote shift
  • Set a < 0 — branches switch to opposite quadrants
  • Increase |a| — branches stretch further from centre
  • Increase |k| — branches squeeze toward the asymptote
