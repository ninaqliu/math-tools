What this shows

This tool explores y = a / [k(x − d)] + c. The base function is y = 1/x — a hyperbola with two branches. Adjust sliders to transform it.

Parameters

  • a — vertical stretch (|a| > 1) or compression (|a| < 1). Negative a reflects both branches.
  • k — horizontal stretch or compression. Affects how quickly the curve approaches the asymptote.
  • d — horizontal shift. Moves the vertical asymptote left or right.
  • c — vertical shift. Moves the horizontal asymptote up or down.

Key concepts

  • Vertical asymptote: x = d — the curve never crosses this line
  • Horizontal asymptote: y = c — the curve approaches but never reaches this
  • Two branches — one on each side of the vertical asymptote
  • Domain: all x ≠ d  ·  Range: all y ≠ c
  • Centre of the hyperbola is always at (d, c)

Try this

  • Move d — watch the vertical asymptote shift
  • Move c — watch the horizontal asymptote shift
  • Set a < 0 — branches flip to opposite quadrants
  • Increase |a| — branches stretch further from centre
  • Increase |k| — branches compress closer to asymptote
