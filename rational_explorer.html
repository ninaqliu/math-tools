What this shows

This tool explores the rational (reciprocal) function. The numerator is a and the denominator is k(x − d), producing a hyperbola with two branches.

Parameters

  • a — scales the numerator. Larger |a| = branches further from centre. Negative a flips both branches.
  • k — scales the denominator. Larger |k| = branches compress toward the asymptote.
  • d — shifts the vertical asymptote left or right.
  • c — shifts the horizontal asymptote up or down.

Key concepts

  • Vertical asymptote: x = d — denominator = 0 here, curve undefined
  • Horizontal asymptote: y = c — curve approaches this as x → ±∞
  • Two branches, one in each pair of opposite quadrants from centre (d, c)
  • Domain: all x ≠ d  ·  Range: all y ≠ c

Try this

  • Move d — watch the vertical asymptote move
  • Move c — watch the horizontal asymptote move
  • Set a < 0 — branches switch quadrants
  • Increase |a| — branches stretch further
  • Increase |k| — branches squeeze toward the asymptote
