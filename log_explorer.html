Logarithmic Functions

y = a · logb(x)

Read me first What this tool shows & how to use it ▾

Three modes

  • log base b — explore any base. Adjust a and b sliders.
  • log₁₀ — common logarithm (base 10). Only a slider active.
  • ln — natural logarithm (base e ≈ 2.718). Only a slider active.

Parameters

  • a — vertical stretch/compression. Negative a flips the curve below the x-axis.
  • b — base (log base b mode only). b > 1 = increasing curve, 0 < b < 1 = decreasing. b ≠ 1.

Key concepts

  • Domain: x > 0 only — log of a negative number is undefined
  • Vertical asymptote: x = 0 — the curve approaches but never crosses the y-axis
  • Always passes through (1, 0) since log of 1 = 0 for any base
  • Red dot = (1, 0)  ·  Green dot = (b, a) key point  ·  purple dashed line = vertical asymptote x = 0

Try this

  • In base b mode: set b = 10 — matches the log₁₀ tab exactly
  • Set b = 2.718 ≈ e — matches the ln tab closely
  • Set 0 < b < 1 — which direction does the curve go?
  • Make a negative — how does the curve flip?
