Three modes

  • log base b — explore any base from 0.01 to 10. Use a and b sliders.
  • log₁₀ — common logarithm (base 10). Only a slider needed.
  • ln — natural logarithm (base e ≈ 2.718). Only a slider needed.

Parameters

  • a — vertical stretch/compression. Negative a flips the curve.
  • b — base (only in log base b mode). b > 1 = increasing curve, 0 < b < 1 = decreasing curve. b cannot equal 1.

Key concepts

  • Domain: x > 0. Vertical asymptote always at x = 0.
  • Always passes through (1, 0) since log of 1 = 0 for any base.
  • Red dot = (1, 0)  ·  Green dot = (b, a) key point.
  • log₁₀(10) = 1  ·  ln(e) = 1  ·  logb(b) = 1

Try this

  • In base b mode: set b = 10 — matches log₁₀ tab exactly
  • Set b = 2.72 ≈ e — matches ln tab closely
  • Set b between 0 and 1 — what direction does the curve go?
  • Make a negative — how does the curve flip?
