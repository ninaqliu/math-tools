What this shows

This tool explores: y = a · logb(x) where b is the base and a is a vertical multiplier. Adjust sliders to see how each changes the graph.

Parameters

  • a — vertical stretch/compression. If a < 0 the curve flips below the x-axis.
  • b — the base (b > 0, b ≠ 1). Controls how fast the curve grows. b > 1 = increasing, 0 < b < 1 = decreasing.

Key concepts

  • Domain: x > 0 (log is only defined for positive x)
  • Vertical asymptote always at x = 0
  • Always passes through (1, 0) since logb(1) = 0 for any base
  • Always passes through (b, a) since logb(b) = 1
  • log10(x) = common log · ln(x) = natural log (base e ≈ 2.718)

Try this

  • Set b = 10 — this is the common logarithm (log₁₀)
  • Set b ≈ 2.718 — this is the natural log (ln)
  • Set b = 2 — how many times do you double to reach x?
  • Make a negative — which way does the curve flip?
  • What happens as b gets closer to 1?
