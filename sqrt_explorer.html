What this shows

This tool explores y = a√[k(x − d)] + c. The base function is y = √x — a half-parabola starting at the origin. Adjust sliders to transform it.

Parameters

  • a — vertical stretch (|a| > 1) or compression (|a| < 1). Negative a reflects the curve below the x-axis.
  • k — horizontal stretch (|k| < 1) or compression (|k| > 1). Negative k reflects the curve left (domain flips).
  • d — horizontal shift. Moves the starting point left or right.
  • c — vertical shift. Moves the curve up or down.

Key concepts

  • Domain: x ≥ d (when k > 0) — square root requires non-negative input
  • Starting point (endpoint) is always at (d, c) — marked with red dot
  • Faint ghost curve shows base y = √x for reference
  • Negative k flips the domain: curve goes left instead of right

Try this

  • Set a < 0 — curve flips downward
  • Move d right — starting point shifts right
  • Move c up/down — whole curve shifts vertically
  • Set k < 0 — what happens to the direction of the curve?
