Square Root Functions

y = a√[k(x − d)] + c

Read me first What this tool shows & how to use it ▾

What this shows

This tool explores the square root function with four transformations. The radical sign (vinculum) covers the full expression k(x − d) inside.

Parameters

  • a — vertical stretch/flip. Negative a reflects the curve below the x-axis.
  • k — horizontal stretch/compression. Negative k flips the domain direction so the curve extends left.
  • d — horizontal shift. Moves the starting point left or right.
  • c — vertical shift. Moves the whole curve up or down.

Key concepts

  • Domain: k(x − d) ≥ 0 — the expression under the radical must be non-negative
  • Starting point (endpoint) is always at (d, c) — marked with a red dot
  • Faint grey curve shows the base y = √x for reference
  • k < 0 flips the domain: curve extends left instead of right

Try this

  • Set a < 0 — curve reflects below the x-axis
  • Move d — the starting point shifts horizontally
  • Move c — the whole curve shifts up or down
  • Set k < 0 — what direction does the curve extend?
