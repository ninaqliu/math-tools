Present & Future Value

PV = FV / (1+r)ᵗ  ·  FV = PV × (1+r)ᵗ

Read me first Time value of money — the core idea ▾

The core idea

  • Money today is worth more than the same amount in the future — because today's money can be invested and earn returns
  • This is called the Time Value of Money

Present Value (PV)

  • Asks: "How much do I need to invest today to reach a target future amount?"
  • The higher the rate or the longer the time, the lower the PV (you need less today)
  • PV is a decreasing function of time — money needs discounting

Future Value (FV)

  • Asks: "How much will my money grow to after t years?"
  • The higher the rate or the longer the time, the higher the FV
  • FV is an increasing exponential function of time

They are exact inverses

  • PV × (1+r)ᵗ = FV — always
  • Switch tabs to see both perspectives with the same numbers

Try this

  • PV: set FV=$10,000, r=5%, t=10 — how much to invest today?
  • FV: set PV=$5,000, r=7%, t=10 — what does it grow to?
  • Increase r — PV drops, FV rises. Why?
