Quadratic Functions
y = a(k(x − d))² + c
▾
Transform parabolas using four parameters.
  • a — vertical stretch and reflection
  • k — horizontal stretch
  • d — horizontal shift (moves vertex left/right)
  • c — vertical shift (moves vertex up/down)
